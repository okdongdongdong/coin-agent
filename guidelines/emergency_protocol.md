# 비상 정지 프로토콜

## 서킷브레이커 조건
다음 중 하나라도 충족되면 즉시 모든 거래 중단:

1. **일일 손실 한도**: 30,000 KRW (시작 자본의 10%)
2. **총 손실 한도**: 90,000 KRW (시작 자본의 30%)
3. **연속 손실**: 5회 연속 손실 거래
4. **킬스위치**: `data/KILL` 파일 존재 시 즉시 정지

## 서킷브레이커 발동 시 행동
1. 모든 미체결 주문 취소
2. 신규 주문 제출 금지
3. 상태를 `CIRCUIT_BREAKER_TRIPPED`로 변경
4. `data/circuit_breaker.json`에 발동 사유 기록
5. 로그에 경고 메시지 출력

## 수동 킬스위치
```bash
# 즉시 정지
python -m coin_agent stop
# 또는
touch data/KILL
```

## 복구 절차
1. `data/KILL` 파일 삭제
2. `data/circuit_breaker.json` 확인 및 원인 분석
3. 필요 시 설정 조정
4. 봇 재시작
