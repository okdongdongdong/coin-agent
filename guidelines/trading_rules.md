# 거래 규칙

## 기본 설정
- 시작 자본: 300,000 KRW
- 거래소: 빗썸 (Bithumb)
- 기본 마켓: KRW-BTC
- 기본 모드: Paper Trading (dry_run=true)

## 포지션 규칙
- 단일 에이전트 최대 포지션: 전체 자본의 40%
- 최소 KRW 보유량: 50,000 KRW (긴급 대비)
- 주문당 최소 금액: 5,000 KRW (빗썸 최소 주문)
- 주문 쿨다운: 60초

## 매매 규칙
- 지정가 주문만 사용 (시장가 미지원)
- 매수 시 현재가 대비 -0.05% 오프셋
- 매도 시 현재가 대비 +0.05% 오프셋
- 수수료율: 0.25%

## 시그널 해석
- confidence 0.7 이상: 강한 시그널 → 즉시 실행
- confidence 0.5~0.7: 보통 시그널 → 다른 에이전트 시그널과 종합 판단
- confidence 0.5 미만: 약한 시그널 → 무시

## 자본 배분
- 에이전트당 최소 배분: 5%
- 에이전트당 최대 배분: 40%
- 리밸런싱 주기: 50틱마다
- 워밍업 기간: 최소 10거래 완료 후 점수 반영
